# Meeting Notes

Weekly meeting notes go here.

## File Naming

Use this format: `YYYY-MM-DD.md`

Example: `2025-02-12.md`

## Template

Copy this for each meeting:

```markdown
# Meeting Notes - YYYY-MM-DD

**Attendees:** Name1, Name2, Name3, Name4

## Last Week Review

- [ ] Task 1 - Owner - Status
- [ ] Task 2 - Owner - Status

## Discussion

### Topic 1
Notes here

### Topic 2
Notes here

## Blockers

- Blocker description - who needs help

## Action Items (Next Week)

| Task | Owner | Due |
|------|-------|-----|
| Description | Name | Date |

## Next Meeting

Date: YYYY-MM-DD
```
