# Notebooks

Jupyter notebooks for exploration and analysis.

## Naming Convention

Number our notebooks in order:
- `01_data_inspection.ipynb`
- `02_eda.ipynb`
- `03_feature_engineering.ipynb`
- `04_baseline_model.ipynb`
- etc.

## Before Committing

**Clear all outputs** before committing to keep diffs readable:
- Kernel → Restart Kernel and Clear All Outputs
- Then save and commit

## Ownership

Add your name to notebooks you're working on:
- `02_eda_eugene.ipynb` (if multiple people do EDA)
